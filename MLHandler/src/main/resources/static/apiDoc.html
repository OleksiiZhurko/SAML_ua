Processes requests to predict the tone of news headlines in Ukrainian.

Queries

models

Description

Query to list all available ML models

Response

Returns [MLModel!]!

Example

Query
query Models {
  models {
    name
    description
  }
}
Response
{
  "data": {
    "models": [
      {
        "name": "abc123",
        "description": "abc123"
      }
    ]
  }
}
Queries

processTexts

Description

Query to process texts using an ML model

Response

Returns [ProcessedText!]!

Arguments
Name Description
input - ProcessTextsInput!

Example

Query
query ProcessTexts($input: ProcessTextsInput!) {
  processTexts(input: $input) {
    model
    text
    predicted
    probabilities {
      positive
      neutral
      negative
    }
  }
}
Variables
{"input": ProcessTextsInput}
Response
{
  "data": {
    "processTexts": [
      {
        "model": "abc123",
        "text": "xyz789",
        "predicted": "xyz789",
        "probabilities": Probabilities
      }
    ]
  }
}
